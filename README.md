\# Maritime Management System



Full-stack web application using React and Django.



\## Tech Stack

\- Frontend: React + Tailwind

\- Backend: Django + DRF

\- Database: SQLite

\- Real-time: Channels + Redis



\## Setup



\### Backend

cd backend

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver



\### Frontend

cd frontend/vessel-frontend

npm install

npm start
